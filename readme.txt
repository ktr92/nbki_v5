1. общий момент - все блоки, даже похожие на старые по виду, на самом деле обновлены. Изменены размеры шрифтов, отступы, кнопки, размеры блоков. 
Все это требует А) добавления новых стилей, классов (в минимальных случаях),  
Б) либо чаще всего разработки новых блоков с нуля т.к. некоторые из "старых" блоков остаются на других страницах и нужно не зацепить их в случае изменения по новым макетам.
Поэтому многие блоки пришлось делать заново с 0.
